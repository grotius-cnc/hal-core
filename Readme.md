Install guide:

    https://github.com/grotius-cnc/hal_core/blob/main/src/readme.md

To startup the hal environment:

    $ cd scripts
    $ . ./rip-environment
    
To test if hal is working:

    $ cd bin 
    $ ./halcmd show
